## 權證基本認識

台股權證是由證券商發行的衍生性商品，常見分為：

- 認購權證：看漲標的股票或指數
- 認售權證：看跌標的股票或指數

權證不是股票，會受到下列因素影響：

- 標的股價格
- 履約價
- 到期日
- 剩餘天數
- 隱含波動率
- 發行商報價與流動量
- 價內、價平、價外程度

## 權證核心風險

1. 時間價值會流失，到期日越近壓力越大。
2. 槓桿會放大獲利，也會放大虧損。
3. 價外權證可能最後變成沒有價值。
4. 報價不一定永遠漂亮，買賣價差會吃掉獲利。
5. 不能只看價格便宜，很多便宜權證是因為到期近、價外深、風險高。

## 權證只允許的使用方式

- 只能用小資金試單，不用主要資金壓權證。
- 單一權證資金建議不超過總資金 5% 到 10%。
- 進場前必看：剩餘天數、成交量、買賣價差、履約價、隱含波動率。
- 沒有停損價，不進場。
- 不碰快到期、成交量太低、買賣價差太大的權證。
